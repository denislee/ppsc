Schedule

Politeness

Minimum delay between requests to the same site (plus random jitter). Higher = gentler on the sites and less likely to be rate-limited or blocked. The scraper also backs off automatically on HTTP 429/503.

Photos

When enabled, each new listing's detail page is fetched once and its photo gallery is downloaded to local disk (served at /photos). Capped per run so a fresh database doesn't fetch everything at once — the rest are picked up on later scrapes.

Nearest subway station

Each listing's address is geocoded once (via OpenStreetMap Nominatim) to find the closest São Paulo Metrô station, shown on cards and mapped in the detail view. Capped per run so a fresh database doesn't geocode everything at once — the rest are resolved on later scrapes (or instantly when you open a listing).
